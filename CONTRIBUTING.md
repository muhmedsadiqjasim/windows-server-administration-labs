# Contributing

Thank you for your interest in contributing to this project!

This repository contains hands-on Windows Server Administration labs designed for learning and practice. Contributions that improve the quality, accuracy, and clarity of the labs are welcome.

## Ways to Contribute

You can help by:

- Fixing spelling or grammar mistakes.
- Improving explanations or documentation.
- Reporting incorrect or outdated configurations.
- Adding screenshots where they improve understanding.
- Suggesting new lab ideas.
- Improving PowerShell or Windows Server commands.
- Adding verification or troubleshooting steps.

## Before You Start

Before opening an issue or pull request:

- Check if a similar issue or pull request already exists.
- Read the README to understand the project structure.
- Test any configuration before submitting changes.

## Pull Request Guidelines

Please ensure that your pull request:

- Focuses on a single topic or lab.
- Uses proper Markdown formatting.
- Includes clear explanations.
- Keeps screenshots high quality and readable.
- Preserves the existing folder structure.
- Does not include sensitive information such as passwords, product keys, or personal IP addresses.

## Writing Style

Please keep documentation:

- Clear and concise.
- Professional.
- Beginner-friendly.
- Grammatically correct.
- Consistent with Microsoft's terminology.

## Lab Contributions

If you're adding a new lab, include:

- Objective
- Requirements
- Network topology (if applicable)
- Step-by-step implementation
- Verification
- Troubleshooting
- Expected results

## Reporting Issues

When reporting an issue, include:

- The lab name
- The section where the problem occurs
- A description of the issue
- Screenshots or error messages (if available)

## Thank You

Thank you for helping improve this project and making it more useful for everyone learning Windows Server Administration.
