# Security Policy

## Supported Versions

This repository is maintained as a single, continuously updated version. Security fixes and documentation updates are applied to the latest version only.

## Reporting a Vulnerability

Although this repository primarily contains documentation, configuration examples, and lab files for a virtualized Windows Server environment, security-related issues are taken seriously.

If you discover a security issue, such as:

- Credentials or sensitive information accidentally committed to the repository
- Insecure configuration examples that could mislead users in production environments
- Scripts that could cause unintended security risks
- Any other security concern

Please do **not** create a public issue.

Instead, contact the repository maintainer through GitHub or open a private security advisory if available. The issue will be reviewed, and appropriate action will be taken as soon as possible.

## Scope

This project is intended for educational purposes and all labs are designed to be performed in an isolated virtual lab environment. The configurations and examples provided should be reviewed and adapted before being used in a production environment.

Users are responsible for validating all configurations according to their organization's security requirements.
